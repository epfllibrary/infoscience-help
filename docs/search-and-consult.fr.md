# Rechercher et consulter

![Bannière — Rechercher et consulter](assets/images/search-and-consult/banner.png)

Pour trouver une publication, un.e auteur.rice, un laboratoire ou un sujet spécifique, naviguez à travers Infoscience !

*(L'authentification EPFL n'est pas nécessaire)*

---

## Recherche simple

La barre de recherche sur la page d'accueil permet de rechercher un ou plusieurs mots-clés : nom, titre, sujet, date…. Vous trouverez les publications et travaux correspondant à ces critères.

Dans la barre de recherche, saisissez les termes souhaités, puis cliquez sur **Recherche**.

![Barre de recherche Infoscience](assets/images/search-and-consult/BarreDeRecherche.png)

**Afficher les résultats et utiliser les filtres**

Lorsque la liste de résultats s'affiche, vous pouvez :

- l'affiner **à l'aide des filtres** proposés à gauche (Type de document, Auteur.trice, Année de publication, Licence…)
- trier la liste en utilisant les paramètres proposés en bas à gauche (pertinence, date de dépôt, …)
- modifier le nombre de résultats affichés, en bas à gauche

![Filtres et facettes sur la page de résultats](assets/images/search-and-consult/AffichezFiltres.png)

!!! note
    Diverses astuces peuvent faciliter votre recherche ; découvrez-les en consultant les exemples de requêtes expertes dans la section [Création d'une requête avancée](publication-lists.fr.md#construction-dune-requete-avancee).

## Menus de navigation

Les menus permettent de parcourir les contenus selon **plusieurs points d'entrée :**

- Recherche : Facultés et Collèges, Centres et plateformes de recherche
- Éducation : Programmes doctoraux (thèses EPFL), Sections d'enseignement (travaux étudiant.e.s)
- Innovation (brevets)

![Menus de navigation Infoscience](assets/images/search-and-consult/Menus.png)

**Explorer** : pour parcourir les contenus d'Infoscience via des listes ordonnées :

- Travaux académiques (recherche avancée)
- En Open Access
- Membres EPFL
- Unités EPFL
- Revues
- Événements (conférences, etc.)

Le menu **EXPLORER** (**1**) permet de cibler rapidement un périmètre de recherche et de combiner plusieurs critères (Titre, Auteur, Type de document…) avec les opérateurs booléens (ET, OU, SAUF).

## Consulter une notice

Les notices se présentent par défaut sous forme brève. En cliquant sur le titre, vous accédez à la notice détaillée.

**La notice brève (1) présente :**

- Le type de contenu (ici « Publication »)
- Le type d'accès (ici « Open Access »)
- Le titre
- Le publisher (ou container)
- Les premiers auteur.trices (cliquables pour voir leur profil)
- Le début du résumé
- Les indicateurs statistiques de consultation et/ou de téléchargement
- Un widget Altmetrics (si présent)

![Notice brève (1)](assets/images/search-and-consult/NoticeBreve1.png)

**La notice détaillée (2) présente en sus :**

- Le type de document
- La date de dépôt
- Le nom du journal
- Un bouton **Exporter** (formats : RIS, JSON, Cerif-XML, Datacite-XML, CSV et BibTeX)
- Le bouton **Statistiques** (Total visits, top city views, file visits), exportables sous Excel ou CSV
- Un bouton **S'abonner** (pour les utilisateur.trices EPFL connecté.es)
- Le bouton « **…** » pour visualiser toutes les métadonnées de la notice

![Notice détaillée (2)](assets/images/search-and-consult/Capture-decran-2024-08-28-111203.png)

**Dans l'onglet Files, les informations concernant le(s) fichier(s) (3) :**

- Le nom du fichier
- Type/Version du fichier
- Le mode d'accès
- La licence du fichier
- Taille du fichier
- Format du fichier

![Onglet Files — informations sur le fichier (3)](assets/images/search-and-consult/NoticeDetaillee3.png)

**Dans l'onglet Details (4)** : toutes les métadonnées.

![Onglet Details (4)](assets/images/search-and-consult/Capture-decran-2024-08-28-111828.png)

**Dans l'onglet Metrics (5)** : les statistiques de consultation et de téléchargement.

![Onglet Metrics — statistiques (5)](assets/images/search-and-consult/NoticeDetaillee5.png)

## Accéder aux fichiers

Pour accéder au(x) fichier(s) de la notice, cliquez sur le bouton **Accéder aux fichiers**.

![Bouton Accéder aux fichiers](assets/images/search-and-consult/AccederAuxFichiers.png)

## Recevoir des alertes

Infoscience vous permet de recevoir des alertes lors de modifications sur une notice, un profil ou une unité.

**S'abonner à une alerte (1)** depuis la notice ou le profil concerné :

![S'abonner — bouton (1)](assets/images/search-and-consult/Alertes.png)

**Choisir la fréquence** des alertes (quotidienne, hebdomadaire ou mensuelle) (**2**) :

![Choix de la fréquence (2)](assets/images/search-and-consult/Alertes2.png)

**Gérer ou supprimer vos abonnements** depuis votre profil (**3**) :

![Gestion des abonnements (3)](assets/images/search-and-consult/Alertes3.png)

## Consulter les statistiques

**Statistiques du site** : accessibles depuis le menu principal, elles permettent de visualiser les tendances de consultation de l'ensemble de la plateforme.

![Statistiques du site](assets/images/search-and-consult/StatistiquesSite.png)

**Statistiques d'une notice** : accessibles depuis la notice elle-même via le bouton **Statistiques**, elles indiquent le nombre de vues et de téléchargements du fichier associé.

![Statistiques d'une notice](assets/images/search-and-consult/StatistiquesNotice.png)
